# Sapling NVIM

Plugin for working with [Sapling SCM](https://sapling-scm.com/) in NVIM

</div>

## Installation

### Manually

```sh
git clone https://github.com/AdeAttwood/sapling.nvim ~/.config/nvim/pack/bundle/start/sapling.nvim
```
